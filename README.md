## ✨ Funcionalidades

| Función | Descripción |
|---|---|
| 🎓 **Test de Práctica** | 50 preguntas aleatorias de un banco de 220 |
| 📋 **Examen Junio 2025** | Las 50 preguntas del examen oficial UAL (18/06/2025) en orden |
| 📚 **Biblioteca de Repaso** | Resumen organizado por unidades y secciones para revisar antes del examen |
| 🌙 **Modo Oscuro** | Toggle claro/oscuro, persistido en el dispositivo |
| ✅ **Penalización** | Sistema real: 1 error = −1 acierto. Nota sobre 10 |
| 📱 **PWA para iOS** | Instalable en iPhone desde Safari como app nativa |

## 📱 Cómo instalar en iPhone

1. Abre [la URL](https://ualfrv980.github.io/test-seguridad/) en **Safari**
2. Pulsa el botón de compartir ↑
3. Selecciona **"Añadir a pantalla de inicio"**
4. Ya tienes la app instalada sin necesidad de App Store

---

## 🛠️ Tecnología

- **React 18** (via CDN, sin build step)
- **Babel Standalone** (JSX en el navegador)
- **Tabler Icons** (iconografía)
- **CSS Variables** (theming claro/oscuro)
- **localStorage** (persistencia del tema)
- **Service Worker** (PWA, caché offline)
